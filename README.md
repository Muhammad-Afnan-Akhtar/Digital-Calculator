# 🧮 Calculator

> A modern, responsive web-based calculator built with HTML, CSS, and JavaScript.

![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat&logo=javascript&logoColor=black)

---

## ✨ Features

- **Basic Operations** — Addition, subtraction, multiplication, and division
- **Extra Functions**
  - `AC` — Clear all input
  - `DEL` — Delete last character
  - `%` — Percentage calculation
- **Polished Interface** — Dark theme with gradient background, circular button design, orange accent for equals, green for operators

---

## 🚀 Getting Started

1. Clone or download the project
2. Open `index.html` in any web browser
3. Start calculating!

## 🔢 Button Reference

| Button | Function | Type |
|--------|----------|------|
| `0–9`, `00` | Number input | Numeric |
| `.` | Decimal point | Numeric |
| `+` `-` `*` `/` | Arithmetic operators | Operator |
| `%` | Percentage | Operator |
| `=` | Calculate result | Calculate |
| `AC` | Clear all input | Control |
| `DEL` | Delete last character | Control |
